# Bike_Buyers_Sales_Project
Analysis of Bike Byuers Data using Advances Excel tool

Bike Buyers Data Analysis Dashboard (Advanced Excel)
Overview

This project focuses on analyzing customer purchasing behavior using Advanced Excel. The goal was to clean the dataset, perform data analysis, and build an interactive dashboard to identify factors influencing bike purchases and customer demographics.

The project demonstrates practical Excel skills used in real-world business analysis and reporting.

Dataset

The dataset contains information about customers, including:

Age
Gender
Income
Marital Status
Education
Occupation
Region
Bike Purchase Status

The data was analyzed to understand customer segments and buying patterns.

Microsoft Excel

Data Visualization

Project Workflow

1. Data Cleaning
Removed duplicate records.
Handled missing/null values.
Standardized data for analysis.

2. Data Analysis
Analyzed customer demographics and purchase behavior.
Identified trends based on income, age, gender, and occupation.

4. Dashboard Creation
Built Pivot Tables and Pivot Charts.
Added interactive slicers for easy filtering.
Created a user-friendly dashboard to visualize key insights.

Key Insights
Identified customer groups with higher bike purchase rates.
Analyzed the relationship between income and purchasing decisions.
Compared purchase behavior across age groups and genders.
Highlighted demographic trends influencing bike sales.
